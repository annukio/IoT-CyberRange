How to launch this lab (wip):
1. Download repo

2. Make the scripts executable:
```bash
chmod +x launch.sh
```
```bash
chmod +x switch-scenario.sh
```

2. build container for the first challenge:
  ```bash
     ./launch.sh s0
  ```
3. Access http://localhost:3000
4. Play with the terminals!

Current state: scenario 0 and 1 should be launched accordingly.
If you cannot continue s0 (validation check not working) launch the second challenge:

 ```bash
     ./launch.sh s1
  ```
